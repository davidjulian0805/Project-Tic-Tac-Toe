
# Tic-Tac-Toe

This project is a **browser-based Tic-Tac-Toe game** built as part of **The Odin Project JavaScript Course**. The objective of this project is to practice **JavaScript objects, factory functions, modules (IIFE), and DOM manipulation** while organizing code in a clean and modular structure.

The game allows two players to take turns placing their marks on the board until one player wins or the game ends in a tie.

---

## Project Overview

The Tic-Tac-Toe game is structured using multiple JavaScript objects that handle different responsibilities:

* **Gameboard Object** – stores the game board using an array
* **Player Objects** – represent each player
* **Game Controller** – controls the game flow
* **Display Controller** – handles DOM rendering and user interaction

The project was first developed to work **in the browser console**, focusing on game logic before integrating the user interface.

## How the Game Works

1. The **Gameboard object** stores the board as an array.
2. Two **Player objects** are created.
3. Players take turns selecting squares on the board.
4. The game checks for:

   * Winning combinations
   * Tie conditions
5. The result is displayed once the game ends.
6. Players can restart the game using the restart button.
